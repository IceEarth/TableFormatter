# TableFormatter
**Hey**, 
this is a API to get the format for a table in a string: e.g.

                —————————————————————————
                |  name  | age | weight |
                *************************
                | James  | 21  |   75   |
                | Robert | 30  |   84   |
                —————————————————————————
 
You are also able to edit the spacer, the columnLeftBracket, the columnRightBracket, the headerValuesSpacerChar and the lineValuesSpaceChar.
